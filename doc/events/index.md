# 新的起点哦

## 🛫 裁员后的思考
职业生涯才刚刚走过六年，其实是一个关键的节点，心想着正式休息休息，再整理整理自己的工作经验。那时候启动了一个项目，打算用`Vue3 + TS + Vite`来展示，毕竟在之前的公司处理的 `2D 地图`项目比较多，顺便整理整理关于地理坐标和地图的知识。之所以没有一开始选择`VitePress`是因为考虑一个正式的项目比较能够体现地图相关的效果，还弄了个`Github Pages`来展示。但是实际执行下来，想要展示类似知识库的效果，没有成套的开箱即用的样式，总觉得 `投入的时间`和`知识密度`不成正比。🔗 [旧项目](https://lijuncongdiku.github.io/skill-stack-ts/#/)

## 🐈 go2cc (其实是go to see see) 谐音
- 🖊️ 幸亏旧项目也用了大量的markdown文件来作为项目的文档，到时候迁移过来。
- 📖 正如其他页面的扉页中所提及，`go2cc`首要目的是一个人知识库，其次也想记录一些自己的经历和思考
- 🥰 笔者算是一个分享欲比较强的人，从以前开始就会不时写一些随笔，有时候在朋友圈、有时候在云笔记、但这个项目形成雏形的时候，我忽然觉得，学习、工作和生活一起记录的项目应该会更有意思。
