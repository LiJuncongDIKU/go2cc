## 弗洛伊德的三个我
| 名称                | 运作原则 | 核心描述                                                       |
| :------------------ | :------- | :------------------------------------------------------------- |
| **本我 (Id)**       | 快乐     | 代表原始欲望与生物本能，追求即时的满足，不考虑现实后果。       |
| **自我 (Ego)**      | 现实     | 负责协调本我的冲动与超我的克制，寻找在现实世界中可行的方法。   |
| **超我 (Superego)** | 完美     | 代表社会规范、道德准则和理想自我，通过良知和罪恶感来约束行为。 |

&emsp;&emsp;比较赞成这个模型，特别能够解释人困惑、迷茫、或者面临选择时的行为。我们不了解每一个人，但我们可以通过这个模型了解自我。\
&emsp;&emsp;人的行为由超我和本我协调，超我负责做正确的事情，本我负责做想做的事情，自我负责协调二者。
:::tip 强大的自我
- 能够压制本我的冲动
- 不会在超我难以实现的时候过度痛苦（跟自己和解）
- 自我在二者之间波动但又保持在一个合理的水平  ~优雅~
:::

## 社会观
**政治统治阶级的必然性**：根据考古学发现，追溯到至少山顶洞人时期已经出现象征身份的陪葬品。从种植、圈养、群居等角度出发，需要能够管理相对过剩或者季节性收成的食物资源、需要对圈养牲畜作为公共资产进行管理、需要对个体产生约束力的社会规则。而构成这一层管理的，无论是暴力机关、共同的道德约束还是宗教信仰，本质仍然是自上而下的管理——我称之为广义上的政治。

[无政府主义](https://zh.wikipedia.org/wiki/%E6%97%A0%E6%94%BF%E5%BA%9C%E4%B8%BB%E4%B9%89)理论从根本上否定了上述的政治统治阶级的必然性。它并不否认社会规则的必然性，但反对绝对权威的政治体制，任何形式的政治体制都将造成阶级的划分，而社会问题、矛盾和其他公共问题的解决，将依赖于社会的合作和协商，形成非体制化的规则。

以历史的结论来说，无政府主义在世界各地曾经形成浪潮，但绝大多数被打压和消灭。其中也存在正面的例子，例如在作为移民国家的美国，无政府主义者构成的社区自治，成功规避了文化融合——即使州法律与社区法律的冲突，也能维持双方各自安好的认同。无政府主义的天然缺陷包括：
- 强调个人自由，代表团体事务的参与全屏自觉，缺乏约束力，一盘散沙
- 强调协商，一个团体集权则高效，协商则低效，协商能够解决成员参与感问题，但未必能够将团体有效及时事务解决
- 规模问题，在强调个人主义和协商的基础上，无政府主义团体规模太大则面临文化认同、民族认同等问题。因而历史上形成思潮风靡，但政治力量不足的特点。
- 竞争力问题，无政府主义在思潮上以自由和欲望吸引信徒，但其无法有效掌控生产资料，无法解释生产和资源问题，因此注定不可能成为主流政治团体。

### 资本主义的合理性
资本主义（英语：capitalism）是一种生产方式为私有制、其寻求利润构成的经济体系。
> 这个论题比较大了，马克思的《资本论》我才读完第一卷。

但我对资本主义的认同在于与人对生产资料的占有欲望源于社会性动物的占有行为（包括食物、领地、配偶等），对于个体来说有无尽的吸引并产生内驱力。
:::warning 政体与主义
遵循或者号称以资本主义或者社会主义为基础的政治制度，必然会在此上形成独特的政治制度。正如上述的概念，资本主义核心在于解释经济运行的核心逻辑，但对于教育、文化、人权等问题（不同学者对于政体评价的标准不一样）无法解释。但其延伸的政体必须将向一个方向演化：权力的巩固，无论是号称民主灯塔的美国两党党政、还是以德国为典型的政党联合选举模式，其实践结果仍无法摆脱大党专政的本质。而选举本身只能解决民众参与感问题，而不能解决执政者专业性、发展的长期规划、既得利益集团的社会义务等问题——这在后疫情时代尤为突出。
:::

### 马克思主义的正确性
[这里](https://www.marxists.org/chinese/index.html)有一系列社会和哲学相关的著作。

**个人关于其正确性的观点**：马克思称之为历史唯物主义，是以全社会运作过程中产生的问题为核心，以**发展**为目标的社会理论。其可被理解为人文社会在历史发展中是不断变化的，而变化必然产生当前体制无法解决的问题，而发现问题，优化机制是永恒的话题。以此为基础，选择凯恩斯还是哈耶克就不言而喻了。

实践中，改革开放并不是一味地全盘接受市场自由化的理论，而是逐步开放，保持宏观调控的能力，以社会问题为基础：粮票的淘汰、供销社改革解决舌尖上的匮乏问题，优先引入轻工业以充沛劳动力带动经济活性，逐步影响到如电器、汽车、冶金等行业。各行各业似乎融入了完全的自由市场，但实际上应该看待为 **“以自由市场为工具解决人们生活需求和落后生产力的矛盾”** 而并不是“信奉自由市场”。
