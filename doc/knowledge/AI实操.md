## MCP
MCP (Model Context Protocol) 是一种标准化的协议，允许AI调起MCP服务中的工具。
> 场景：我们需要用到大量资料片，比如切图、文档，如果下载、解压、或者复制粘贴的操作喂给AI太费时间了

- [lanhu-mcp](https://github.com/lanhu-ai/lanhu-mcp)：AI 直接读取蓝湖需求文档和设计稿
- [Puppeteer](https://pptr.dev/)：控制浏览器，性能分析、监控或者自动化测试
