# 优化
## SSR / SSG
SSR（Server-Side Rendering）和 SSG（Static Site Generation）都是优化网站性能的重要手段。
| 服务端渲染 | 静态站点生成 |
| ---------- | ------------ |
| 每次请求都在服务器端渲染页面 | 构建时在服务器端渲染页面 |
| 页面内容永远是最新的，适合内容经常变动的场景。 | 页面内容是静态的，适合内容不经常变动的场景。 |
| 响应时间相对长 | 响应时间快 |
| 对服务器负载高 | 对服务器负载低 |
